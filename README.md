# Tareas
1. Quiero poder mostrar la evolución de un pensamiento sobre un tema. O sea, si hoy digo algo, luego lo pienso mejor, eso me gustaría que quede conectado. Nada se pierde, todo se continua, todo suma.
2. Creo que veo dos tipo de post, los de pensamientos sueltos que pueden irse luego encadenando según el pensamiento se va manifestando y los que son reuniones de textos.
